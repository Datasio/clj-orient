clj-orient.script.clojure documentation

Utilities for scripting OrientDB using Clojure.

defdbfn

(defdbfn sym & forms)
Defines a function that can be executed on the server and also from Clojure code.

install-db-fns!

(install-db-fns!)
Installs inside the server all the functions defined with defdbfn.

run-clj!

(run-clj! & forms)
Runs forms as a server-side script and returns the resulting value.
